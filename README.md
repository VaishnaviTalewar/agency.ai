# Agency.ai ✨

A modern, animated agency landing page built with React.js featuring smooth animations, cursor effects, and dark/light mode.

**Live Demo:** https://agency-ai-pi-nine.vercel.app/

## 🚀 Features

- **Dark/Light Mode Toggle** - Seamless theme switching with localStorage persistence
- **Custom Cursor Animation** - Interactive cursor with hover effects
- **Smooth Animations** - Built with Framer Motion for fluid transitions
- **Fully Responsive** - Optimized for mobile, tablet & desktop
- **Modern UI/UX** - Clean design with Tailwind CSS
- **Fast Performance** - Built on Vite for lightning-fast builds

## 🛠️ Tech Stack

- **Frontend:** React.js, Vite
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** React Icons
- **Deployment:** Vercel
